MERGE Example

MERGE Customers AS Target
USING SourceCustomers AS Source
ON Target.CustomerID = Source.CustomerID

WHEN MATCHED THEN
UPDATE SET
Target.Name = Source.Name

WHEN NOT MATCHED THEN
INSERT(CustomerID,Name)
VALUES(Source.CustomerID,Source.Name);
